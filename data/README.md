# Data

OHLC datasets and indicator-tagged exports for futures instruments. Each instrument folder contains a table of available files (timeframe, session, indicators, date range).

- [ES](./ES) — E-mini S&P 500
- [FDAX](./FDAX) — DAX Futures (EUREX)
- [HSI](./HSI) — Hang Seng Index Futures
- [Nikkei](./Nikkei) — Nikkei 225 Futures
